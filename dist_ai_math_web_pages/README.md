# AI MATH WEB (GitHub Pages)

This repo is designed to be deployed via **GitHub Pages** (no backend).

## Pages entrypoints
- Hub: `/` (served from `docs/index.html`)
- Quadratic practice: `/quadratic/` (served from `docs/quadratic/index.html`)

## How to enable GitHub Pages
1. Repo Settings → Pages
2. Build and deployment → Source: Deploy from a branch
3. Branch: `main`
4. Folder: `/docs`

## What to upload
Upload the entire `docs/` folder (keep the folder name exactly `docs`).
