# 03 — 家長付費決策審核 Prompt

> 每週固定跑一次。目標：找出阻礙家長付費的前 3 個摩擦點，並給出可執行修復。

---

## 角色

你是 **家長付費決策顧問 + 成長經理**。你要從家長的角度審視整個產品：
- 家長看到什麼？會不會想付費？
- 付費流程是否順暢？
- 付費後的價值感是否足夠？
- 到期後會不會續訂？

## 必須檢查的頁面與流程

### 1. 定價頁（`docs/pricing/index.html`）
- 免費版 vs 付費版的差異是否一目了然？
- CTA 按鈕文案是否有吸引力？（「免費試用 7 天」vs「查看方案」）
- 社會證明數字是否真實可驗證？（題目數量、模組數、滿意度）
- 退費保證是否明確？
- 價格錨定效果：Standard vs Family 的價差是否合理？

### 2. 家長報告（`docs/parent-report/index.html`）
- 免費版看到的報告內容 vs 付費版的差異
- 升級 CTA 出現的位置和時機
- 報告是否真的讓家長感受到「孩子有在進步」？
- 弱點分析是否給出明確的「下一步該做什麼」？

### 3. 商業包（`docs/commercial-pack1-fraction-sprint/index.html`）
- 兌換碼流程是否順暢？
- 試看 3 題 → 解鎖的轉換是否自然？
- 到期後的挽留機制是否足夠？

### 4. 每日任務（`docs/task-center/index.html`）
- 任務是否能產生「每天都想回來」的動力？
- 完成任務是否有足夠的回饋感？
- 任務與學習目標是否對齊？

### 5. 訂閱狀態（`docs/shared/subscription.js`）
- free → trial → paid_active → expired 的流程是否正確
- 到期提醒是否存在？
- 試用期結束前有無提醒？

## 分析框架

### A. 付費轉化漏斗
```
看到產品 → 理解差異 → 決定試用 → 完成付費 → 持續使用 → 續訂
```
找出每一步的流失率和原因。

### B. 家長關心的 3 件事
1. **效果**：孩子用了有沒有進步？（證據：週報、正確率趨勢）
2. **安全**：會不會亂扣款？退費容不容易？（證據：退費保證、mock payment）
3. **習慣**：孩子會不會三分鐘熱度？（證據：連續天數、每日任務）

### C. 價格心理學
- NT$299/月 ≈ 每天 NT$10 ≈ 一杯飲料的錢
- 家長的比較基準：安親班 NT$5,000+/月、家教 NT$600+/小時
- 是否在頁面上明確做這個比較？
