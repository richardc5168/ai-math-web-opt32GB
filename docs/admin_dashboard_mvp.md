# Admin Dashboard MVP

## Required Panels

1. Platform student overview
2. Teacher/class rollup
3. Pre-test summary across classes
4. Post-test summary across classes
5. Parent report generation status
6. Global before/after summary
7. Teacher usage / class performance trend

## Constraints

- Admin has highest scope.
- Admin endpoints remain explicitly token-gated server-side.
- Dashboard should aggregate rather than expose raw secrets or auth data.
